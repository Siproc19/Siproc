# l10n_gt_hr_payroll_v1

Base funcional/técnica para localización de planilla de Guatemala sobre Odoo 19 Enterprise.

## Incluye
- Campos GT en empleado y contrato
- Parámetros legales por vigencia y empresa
- Horas extra
- Liquidaciones
- Integración base con Payroll / Time Off / Work Entries
- Reporte básico de liquidación

## Importante
Esta es una **v1 base** para acelerar el desarrollo. Antes de usar en producción debes:
- validar criterios legales con asesor laboral/contable en Guatemala
- completar reglas salariales y asientos contables según la empresa
- probar en staging con casos reales
- revisar nombres de vistas heredadas si tu base Odoo tiene cambios por Studio o módulos extra
