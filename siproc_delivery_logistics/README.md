# SIPROC Delivery Logistics

Módulo base para Odoo 19 para gestionar logística de entregas:
- rutas
- vehículos
- pilotos
- geolocalización
- vínculo con ventas y entregas

## Instalación
1. Subir la carpeta `siproc_delivery_logistics` a tu repositorio de Odoo.
2. Actualizar Apps.
3. Instalar el módulo **SIPROC Delivery Logistics**.

## Nota
Este es un MVP base. Puede requerir ajustes finos de vistas XPath o mejoras según tu base Odoo 19 y apps instaladas.
